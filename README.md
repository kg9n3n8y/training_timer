# 筋トレタイマー

トレーニング時間と休憩時間をセット単位で管理できるシンプルな筋トレ用タイマーです。ブラウザのみで動作し、オフラインでも継続して利用できます。

- 公開ページ（例）: GitHub Pages などの静的ホスティングに配置して利用してください
- 技術スタック: HTML / CSS / Vanilla JS / Service Worker

## 主な機能
- トレーニング時間と休憩時間を10秒刻み（各10〜60秒）で設定可能
- セット数は1〜10回、デフォルトは3セット
- 設定値はローカルストレージに保存され、次回アクセス時に自動復元
- 円形タイマーとプログレスリングで経過時間を直感的に表示
- 開始前に3秒のカウントダウンを実施し、フェーズ切り替え時にはバイブレーションで通知
- UIは明るく視認性の高い配色に刷新
- PWA に対応しており、ホーム画面に追加すればオフラインでも利用可能

## 使い方
1. 任意の静的ホスティングにソース一式を配置し、ブラウザでアクセスします。
2. トレーニング時間・休憩時間・セット数をスライダーで調整します（初期値: 30秒 / 30秒 / 3セット）。
3. 丸いタイマー領域または「スタート」ボタンをタップすると3秒カウントダウン後に計測が開始されます。
4. タイマー進行中に再度タップすると即座に停止し、設定済みの値にリセットされます。
5. 設定を初期値に戻したい場合は「設定をリセット」を利用してください。
6. ブラウザのメニューから「ホーム画面に追加」を行うと、PWAとしてインストールできます。

## 開発メモ
- `manifest.webmanifest` と `service-worker.js` を追加し、主要アセットをキャッシュすることでオフライン対応を実現しています。
- 音声ガイドは使用せず、デバイスのバイブレーション API を活用しています（対応端末のみ）。
